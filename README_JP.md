# What's this?

形態素解析器をできる限り簡単に使うことを目的にしたパッケージです。
 
# Setting up

## MeCab system

[ここ](http://taku910.github.io/mecab/)をみてMecabをインストールしてください。


## python library

このコマンドを実行すれば、必要なライブラリがインストールされます。

```
pip install -r requirement.txt
```

## install

```
[sudo] python setup.py install
```
